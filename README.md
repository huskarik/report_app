## Приложение для загрузки отчётов из моего склада

Скачивает информацию об отгрузках, отчётах коммиссионера и возвратах покупателя за указанный период и формирует их
в excel таблицу

Нужно создать в корне `.env` переменные:

`TOKEN_MS` - токен моего склада
`VALID_USERNAME` - имя пользователя для авторизации
`VALID_PASSWORD` - пароль для авторизации
`SECRET_FOR_FORM` - случайный секрет для работы форм
`BASE_URL` - базовый урл сервера, на котором развёрнуто приложение. Указывать без закрывающего `/`, пример http://127.0.0.1:1234

Вариант развёртывания

1. Создать нужную директорию на сервере и скачать проект на сервер с github. 
`git clone https://github.com/huskarik/report_app.git .`

2. Скопировать окружение, создать `.env` и прописать переменные в созданный файл. В контейнер `.env` не попадает
`cp .env.example .env`

3. Сборка проекта через Dockerfile

`docker build -t report_app .`

4. Запуск:
```bash
docker run -d \
  --name flask-report \        # имя контейнера
  -p 5000:5000 \               # порт хоста:порт контейнера
  --env-file .env \            # подхватывает все переменные из .env
  report_app                # имя образа, который ты собрал
```
